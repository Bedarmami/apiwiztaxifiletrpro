# Taxi Filter Cloud Backend

Этот сервер предназначен для работы в паре с приложением **Taxi Filter PRO**. 

## Функции:
* **Облачный интеллект:** Обмен "белыми списками" адресов между всеми водителями.
* **Карта коллег:** Синхронизация местоположения (анонимно).
* **Сбор статистики:** Замена ТГ-бота для хранения данных о заказах.

## Как развернуть на Railway:
1. Создай новый проект на [Railway.app](https://railway.app/).
2. Выбери "Deploy from GitHub repo".
3. Укажи папку `server`.
4. Railway сам определит Node.js и запустит проект.
5. После деплоя возьми полученный URL (например `https://xxx.up.railway.app`) и вставь его в `DriverNetworkManager.kt` в Android проекте.

## API Эндпоинты:
* `POST /api/location` - обновление координат.
* `GET /api/nearby` - список активных водителей.
* `GET /api/intel` - загрузка общих списков ИИ.
* `POST /api/orders` - лог заказа.
